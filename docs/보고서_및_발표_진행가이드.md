## 2. 🚀 내일 보고서 & PPT 피치덱에 반영할 핵심 강점 (Add-on Highlights)

### ① Multi-API 융합 데이터 구축 (3원 ➔ 5원 데이터 확장 로드맵)
* **1축 (시민 수요)**: 상상대로 서울 정책 제안 824건
* **2축 (현장 민원)**: 국민신문고 전국/서울 민원 582건 (OpenProposal API)
* **3축 (정책 공급)**: 몽땅정보통 서울시 공식 사업 323건
* **4축 (공공 통계)**: 서울시 25개 자치구 합계출산율(TFR), 출생아 수, 보육시설 인프라
* **[확장 로드맵 5축]**: **네이버 뉴스 API (언론 여론 트렌드 3중 검증)** + **서울 열린데이터 광장 (우리동네키움센터/어린이집 실시간 정원 API)**

### ② 행정 실무자 직관적 판단을 돕는 UI/UX 가독성 디자인
1. **유사 제안 원스톱 AI 군집화 & 일괄 행정 답변 공문 자동 생성 시스템**:
   * 유사 안건을 묶어 담당 부서명(예: `저출생사업1팀`), 몽땅정보 연계 혜택, 공식 행정 공문 견본 자동 생성 (`BatchReplyModal`)
   * 각 제안 카드별 **`[🔗 상상대로 서울 원문 URL 직접 확인 ↗]`** 검증 링크 제공
2. **국민신문고 연관 민원 Executive 고가독성 모달 (`CivilRequestDetailModal`)**:
   * 처리기관(`보건복지부`, `서울특별시`, `경산시` 등), 접수일, 상태 표출
   * 8대 정책 분류계층 Breadcrumb 경로 시각화 (`출산·산후 초기지원 › 출산가구 초기지원 › 출산지원금`)
   * **`[🔗 국민신문고 포털에서 원문 전체 열람 ↗]`** 1클릭 검증 액션 제공
3. **상위 개념(8대 대분류) ↔ 하위 개념(6대 생애주기) 양방향 동조 필터링**:
   * 대분류 `전체` 클릭 시 생애주기도 자동으로 `전체` 동조 이동하여 사용자 혼선 차단

## 3. 🎯 데모데이 라이브 시연 시나리오 (10분 발표 동선)

1. **문제 제기 (1분)**: 서울시 출산/보육 민원과 지원 정책이 부서별로 파편화되어 정책 공백 탐지가 어려움.
2. **해결 책 소개 (2분)**: UKKKK 대시보드 (수요-민원-정책-통계 4원 매핑).
3. **라이브 시연 (5분)**:
   * [홈 대시보드]: 25개 자치구 지도 클릭 ➔ 미답변 비율 & 긴급 키워드 팝업
   * [부서별 접수현황]: 상위 대분류 ↔ 생애주기 동조 탭 전환 ➔ 유사 제안 일괄 답변 공문 생성 시연
   * [국민신문고 교차검토]: 1클릭 모달 열람 ➔ 8대 카테고리 계층 및 국민신문고 포털 원문 직접 확인
4. **결론 및 확장성 (2분)**: 네이버 뉴스 API & 실시간 키움센터 인프라 API 연동을 통한 지자체 정책 의사결정 고도화 비전.
