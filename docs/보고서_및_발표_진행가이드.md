## 2. 🚀 내일 보고서 & PPT 피치덱에 반영할 핵심 강점 (Add-on Highlights)

### ① Multi-API 융합 데이터 구축 완료 (5원 데이터 통합 완결)
* **1축 (시민 수요)**: 상상대로 서울 정책 제안 824건
* **2축 (현장 민원)**: 국민신문고 전국/서울 민원 582건 (OpenProposal API)
* **3축 (정책 공급)**: 몽땅정보통 서울시 공식 사업 323건
* **4축 (공공 통계)**: 서울시 25개 자치구 합계출산율(TFR 0.630명, 2025 잠정 속보치), 출생아 수(45,500명, 2025 잠정 속보치), 보육시설 인프라
* **5축 (언론 보도)**: 네이버 뉴스 API 실시간 검색 + NMF 토픽 모델링 연구 CSV 융합 뉴스 DB 1,145건 연동 (이슈강도 상/중/하 및 활용유형 메타 태그 완료)

### ② 행정 실무자 직관적 판단을 돕는 UI/UX 가독성 디자인
1. **의사결정 갭 분석 매트릭스 대시보드 (Gap Matrix Dashboard)**:
   * 시민수요, 현장민원, 정책공급, 뉴스보도를 8대 카테고리별로 상호 대조하여 정책 공백 가능성을 자동으로 탐지하고 5대 추천 액션을 도출하는 6대 분석 매트릭스 화면 구축.
2. **유사 제안 원스톱 AI 군집화 & 일괄 행정 답변 공문 자동 생성 시스템**:
   * 유사 안건을 묶어 담당 부서명(예: `저출생사업1팀`), 몽땅정보 연계 혜택, 공식 행정 공문 견본 자동 생성 (`BatchReplyModal`)
   * 각 제안 카드별 **`[🔗 상상대로 서울 원문 URL 직접 확인 ↗]`** 검증 링크 제공
3. **국민신문고 연관 민원 Executive 고가독성 모달 (`CivilRequestDetailModal`)**:
   * 처리기관, 접수일, 상태 표출 및 계층별 Breadcrumb 경로 시각화.
   * **`[🔗 국민신문고 포털에서 원문 전체 열람 ↗]`** 1클릭 검증 액션 제공
4. **HoverScrollText 전광판 스크롤**:
   * 대시보드 내 잘리는 텍스트에 마우스 호버 시 3초 멈춤 가로 스크롤 애니메이션 작동으로 정보 전달력 극대화.

## 3. 🎯 데모데이 라이브 시연 시나리오 (10분 발표 동선)

1. **문제 제기 (1분)**: 서울시 출산/보육 민원과 지원 정책이 부서별로 파편화되어 정책 공백 탐지가 어려움.
2. **해결 책 소개 (2분)**: UKKKK 대시보드 (수요-민원-정책-통계-뉴스 5원 정규화 매핑).
3. **라이브 시연 (5분)**:
   * [홈 대시보드]: 25개 자치구 지도 클릭 ➔ 2025년 최신 잠정 출생아 수 반동 통계 차트 및 미답변 비율, 최다 제안 분야 TOP 3, 최고 공감 제안 TOP 3.
   * [부서별 접수현황]: 상위 대분류 ↔ 생애주기 동조 탭 전환 ➔ 유사 제안 일괄 답변 공문 생성 시연
   * [국민신문고 교차검토]: 1클릭 모달 열람 ➔ 8대 카테고리 계층 및 국민신문고 포털 원문 직접 확인
   * [의사결정 갭 분석 매트릭스]: 시민수요/민원/공급/뉴스보도의 양적 수치를 비교하고, 정책 공백 위험과 🚨 이슈강도, 5대 추천 액션을 한눈에 보는 의사결정 시각화 시연
4. **결론 및 확장성 (2분)**: 5원 데이터 융합을 통한 지자체 정책 의사결정 고도화 비전.
