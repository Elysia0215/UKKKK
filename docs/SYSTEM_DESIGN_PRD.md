# 🏛️ 서울시 출산·양육 정책 의사결정 지원 대시보드 (PRD & 시스템 기획서)

> **최종 한 줄 정의**  
> 본 서비스는 **상상대로 서울 시민 제안(824건)**, **국민신문고 현장 민원(582건)**, **몽땅정보통 정책사업(323건)**, **자치구 공공 통계**를 하나의 8대 정책 분류체계로 통합하여 **시민 수요와 정책 공급의 간극(Gap)을 분석**하고, **18개 실무부서 R&R 1·2·3순위 라우팅 및 원스톱 행정 답변 공문 생성을 제공하는 의사결정 지원 대시보드**입니다.

---

## 1. 프로젝트 배경 및 문제 정의 (Problem Statement)

### 🚨 현장의 문제점
* **데이터 파편화**: 공무원 실무자가 정책을 수립하거나 개선할 때, 시민들의 정책 제안(상상대로), 현장 민원(국민신문고), 기존 서울시 사업(몽땅정보통)이 각각 분산되어 있어 종합적인 상황 판단이 어렵습니다.
* **수요-공급 불일치**: 시민들의 고충이나 불만이 계속 제기됨에도 불구하고, 이것이 "신규 정책이 없어서"인지, "기존 정책의 홍보/신청 절차 문제"인지 명확히 파악하기 힘듭니다.
* **행정 소모도**: 수천 건의 파편화된 제안/민원 낱개 데이터를 일일이 열어보는 것은 과도한 행정력 낭비를 초래합니다.

---

## 2. 해결 방안: Multi-API 융합 수요-공급-민원 갭(Gap) 분석 시스템

본 시스템은 4가지 데이터 원천을 하나의 통합 8대 대분류 체계로 정규화한 후, **동일 주제별 [통합 이슈(Integrated Issue Card)] 단위로 묶어서 제공**합니다.

```
[ 1. 상상대로 시민제안 ] ➔ 시민이 바라는 "아이디어 / 정책 요구" (수요 824건)
[ 2. 국민신문고 민원 ]   ➔ 시민이 겪는 "현장 고충 / 불편 신호" (시급성 582건)
[ 3. 몽땅정보통 정책 ]   ➔ 공공이 제공 중인 "323개 현행 혜택" (공급 323건)
[ 4. 자치구 공공 통계 ] ➔ 서울시 25개 자치구 출생아수 & 보육시설 정원 (인프라 갭)
           ↓
 ── [통합 정책 분류 체계 (8대 대분류)] ──
           ↓
 ── [담당 부서 1·2·3순위 매칭 & 1클릭 일괄 행정 답변 공문 표출] ──
```

## 3. 4대 데이터 원천별 역할 정의

| 데이터 소스 | 주요 역할 | 대시보드 활용 방식 |
|---|---|---|
| **상상대로 시민 제안** | 시민 아이디어 및 정책 개선 요구 (824건) | 제안 수, 공감수, 댓글수, 100% 웹 원문, `[🔗 원문 URL 직접 확인 ↗]` |
| **국민신문고 민원** | 현장 고충, 불만, 시급성 높은 불편 신호 (582건) | 민원 수, OpenProposalItem 풀 본문, `[🔗 국민신문고 포털 열람 ↗]` |
| **몽땅정보통 정책** | 기존 서울시 정책 공급 데이터 (323건) | 사업명, 혜택 요약, `https://umppa.seoul.go.kr` 신청 포털 연동 |
| **자치구 공공 통계** | 서울시 25개 자치구 TFR, 출생아수, 보육시설 | 지도(Choropleth), 이중축 그래프, 인프라 갭 지수 |
| **담당 부서/업무분장** | 정책 수립 및 개선 실행 주체 (18개 실무팀) | 1·2·3순위 부서 후보, 전화번호, 업무 근거 |

## 4. 실무 공무원 4대 주요 업무 시나리오

1. **신규 정책 검토 모드**
   * 시민 제안과 현장 민원은 많지만 관련 기존 정책(몽땅정보통) 매칭이 낮은 경우
   * ➔ **`정책 공백 가능성 탐지`** ➔ **`[추천 액션: 신규 정책 검토]`**
2. **기존 정책 개선·홍보 모드**
   * 민원과 불만은 지속되지만 관련 기존 정책이 이미 존재하는 경우
   * ➔ **`접근성/홍보 문제 가능성`** ➔ **`[추천 액션: 기존 정책 홍보 강화 또는 신청·이용 기준 개선]`**
3. **일괄 행정 공문 답변 모드**
   * 유사 제안 묶음 선택 ➔ 1클릭 **`원스톱 일괄 답변 ↗`** 클릭 ➔ 공문 양식 자동 자동 복사
4. **부서별 맞춤 업무 지원 모드**
   * 담당부서(예: `건강임신지원팀`, `저출생사업1팀`)별로 관련 제안, 민원, 정책을 하나의 카드 단위로 묶어 자동 1차 라우팅 제공.
