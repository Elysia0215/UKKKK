# UKKKK 대시보드 — 학술 논문 및 보고서 근거 (실증 흐름 포함)

> 대시보드의 5대 진단 축(수요·공백·시급성·실행성·신뢰도) 및 우선순위 점수 알고리즘을 뒷받침하는 주요 학술 논문과 보고서를 정리합니다.
> 각 논문은 **가설 → 검증 → 결과 → 결론** 4단계 실증 흐름으로 기술하며, 대시보드 방법론과의 연계를 명시합니다.

## 1. 방법론적 타당성 — 텍스트마이닝 기반 정책 수요 도출

### 오신휘·김혜진 (2020)

**논문:** 텍스트마이닝 기법을 활용한 '저출산' 관련 연구동향 분석: 정부의 저출산 정책 추진 과정과의 비교를 중심으로  
**학술지:** 보건사회연구, 40(3), 492–533

| 단계 | 내용 |
|------|------|
| **가설** | 텍스트마이닝 및 동시출현단어 네트워크 분석이 저출산 분야의 대규모 비정형 텍스트를 체계적으로 분류·분석하는 데 유효한 방법론일 것이다. |
| **검증** | 2006~2020년 발표된 저출산 관련 학술 논문 752편을 수집, 텍스트마이닝 및 동시출현단어 네트워크 분석을 수행하고 정부의 저출산 정책 추진 시기별 연구 키워드 변화를 비교. |
| **결과** | 정책 시기별로 학술 연구의 핵심 키워드 군집이 뚜렷하게 구분되었으며, 텍스트마이닝이 정책 변화와 연구 동향 간의 구조적 관계를 정량적으로 포착하는 데 유효함을 확인. |
| **결론** | 비정형 텍스트 기반의 자동 분류·분석이 저출산 정책 연구에서 실무적 타당성을 갖추었으며, 정책 키워드 변화를 추적하는 모니터링 도구로 확장 가능. |

**🔗 대시보드 연계:** 시민 제안·국민신문고 민원을 NLP 클러스터링으로 8대 분류 체계에 자동 배정하는 본 시스템의 텍스트마이닝 방법론 채택 근거.

## 2. 정책 수요(Demand) 및 보육 인프라 연계

### 성낙일·박선권 (2012)

**논문:** 우리나라 지역별 자녀 양육환경과 출산율에 관한 실증분석  
**학술지:** 한국인구학, 35(2), 73–101

| 단계 | 내용 |
|------|------|
| **가설** | 지역 단위 보육 인프라의 공급 규모가 합계출산율을 제고하는 유의미한 경제적 기여를 할 것이다. |
| **검증** | 전국 232개 시군구의 2009년 횡단면 자료를 대상으로 다중 회귀 분석(Regression Analysis) 수행. |
| **결과** | 보육시설 접근성 및 공급 밀도가 지역 합계출산율에 통계적으로 유의미한 양(+)의 효과(p < 0.05)를 나타냄을 실증. |
| **결론** | 자치구 단위의 보육 인프라 불균형 지점에 국공립 및 돌봄 자원을 신속 확충하는 공급 정책 개입이 타당함. |

**🔗 대시보드 연계:** 자치구별 보육시설 수와 시민 수요 간 미스매치를 비교 분석하여 갭 점수를 산출하는 기초 근거.

## 3. 정책과 대중인식 간 괴리 — 정책 공백(Gap) 지수 산정

### 배기련·문현정·이재일·박미나·박아름 (2021)

**논문:** 텍스트 마이닝을 활용한 저출산 정책과 대중인식 비교  
**학술지:** 디지털융복합연구, 19(12), 29–42

| 단계 | 내용 |
|------|------|
| **가설** | 정부의 저출산 대응정책과 대중이 체감하는 핵심 쟁점 사이에 구조적 괴리가 존재할 것이다. |
| **검증** | 제1~4차 저출산·고령사회 기본계획 정책문서와 제3·4차 기본계획 발표 직후 2주간 뉴스 댓글을 대상으로 빈도분석, 동시출현단어 분석, 구조적 등위성(CONCOR) 분석 실시. |
| **결과** | 정책은 시기별 단절적 변화를 보인 반면, 대중 댓글에서는 '결혼'·'출산' 관련 쟁점이 연속적으로 유지됨. 특히 **주거와 고용** 영역에서 정책과 대중인식 간 격차가 가장 두드러짐. |
| **결론** | 정책 설계 시 대중 체감 수요를 상시 모니터링하여 정책 공백을 진단하는 메커니즘이 필요. |

**🔗 대시보드 연계:** 시민이 체감하는 정책 사각지대와 실제 정책 커버리지 간 괴리를 정량화하는 정책 공백(Gap) 지수 설계의 이론적 근거.

## 4. 자치구 지도 시각화 — 공간 분석 설계

### 육아정책연구소 (KICCE, 2023)

**보고서:** 2023 영유아 주요 통계 (연구자료 ES2401)  
**발행:** 육아정책연구소

| 단계 | 내용 |
|------|------|
| **목적** | 전국 시군구 단위의 영유아 인구, 보육시설 현황, 이용률 등을 종합 정리하여 지역별 보육 수요-공급 불균형을 통계적으로 진단. |
| **방법** | 통계청 인구동향조사, 보육통합정보시스템, 교육통계 등 공공 데이터를 결합하여 지역별 영유아 인구 대비 보육 인프라 지표를 산출. |
| **결과** | 지역 간 보육시설 접근성 및 이용률에 뚜렷한 편차가 존재하며, 영유아 밀집 지역과 보육 인프라 용량 간 불일치가 확인됨. |
| **결론** | 자치구 수준의 수요-공급 갭 모니터링 체계가 필요하며, 공간적 자원 재배분의 근거 자료로 활용 가능. |

**🔗 대시보드 연계:** 서울시 자치구별 수요-공급 갭 히트맵 시각화 설계의 타당성 근거.

## 5. 우선순위 점수(Priority Score) 산출

### 박미경 (2022)

**논문:** 저출산 대응정책의 요구도 및 우선순위 분석: MZ세대 인식을 중심으로  
**학술지:** 한국정책연구, 22(1)

| 단계 | 내용 |
|------|------|
| **가설** | MZ세대가 지각하는 저출산 대응정책의 요구도에는 영역 간 우선순위 차이가 존재할 것이다. |
| **검증** | MZ세대 대상 설문조사를 실시하고 Borich 수요도 분석 및 IPA(중요도-수행도) 분석을 수행하여 정책 영역별 요구도 우선순위를 도출. |
| **결과** | Borich 분석 결과 **자녀양육지원(1순위) > 출산지원(2순위) > 일·가정양립 지원(3순위)** 순으로 요구도가 높게 나타남. IPA에서는 출산지원과 일·가정양립이 '우선 개선' 영역으로 도출. |
| **결론** | 저출산 대응 정책 설계 시 양육지원·출산지원·일·가정양립의 세 축을 핵심 가중치로 반영한 우선순위 모형이 타당. |

**🔗 대시보드 연계:** 시민 제안의 공감수(vote_score) 및 카테고리별 시급성을 핵심 가중치로 조합하여 우선순위 점수를 산출하는 알고리즘 설계의 실증 근거.

## 6. 다부서 파편화 해결 및 부서 자동 라우팅

### 국회예산정책처 (NABO, 2025)

**보고서:** ① 주거지원사업 종합평가: 저출생 대응을 위한 주거정책의 방향과 과제 / ② 일·생활 균형 지원정책 평가  
**발행:** 국회예산정책처, 2025년 7월

| 단계 | 내용 |
|------|------|
| **목적** | 저출생 대응 재정사업의 집행 효율성과 정책 전달체계의 파편화 문제를 분석·평가. |
| **방법** | 저출생 관련 재정사업(주거지원·일생활균형 등)의 예산 집행 구조, 부처 간 역할 분담, 수혜자 전달 경로를 종합 분석. |
| **결과** | 저출생 대응 재정 투자가 2006년 1.0조 원에서 2021년 42.9조 원으로 확대되었으나, 사업이 여러 부처·부서로 분절되어 정책 목표와의 정합성이 부족하고 예산 내 사업 내용의 연속성이 미확보됨을 지적. |
| **결론** | 산재된 유사 정책을 통합 분류하고 부서 간 협업 창구를 마련하여 정책 전달 효율성을 제고하는 컨트롤 타워 기능이 시급. |

**🔗 대시보드 연계:** 서울시 18개 부서에 흩어진 정책을 8대 분류로 통합하고, 관련 담당 부서(R&R)를 1·2·3순위로 자동 라우팅하는 대시보드 도입의 정책적 당위성.

## 7. 시민 민원 기반 돌봄 정책 수요 분석 ⚠️

### 홍향희·이정화 (2026)

**논문:** (제목 미확인 — 학술 데이터베이스 미색인 상태)  
**학술지:** 한국가족자원경영학회지 (추정)

| 단계 | 내용 |
|------|------|
| **가설** | 현금 지원보다 보육 인프라 편의성(시간제 보육, 돌봄 공백 해소) 요구가 민원의 주류 쟁점일 것이다. |
| **검증** | 국민신문고 접수 저출산 민원 대상 CONCOR 및 텍스트마이닝 클러스터링. |
| **결과** | 단순 수당 불만보다 맞벌이 돌봄 공백, 시간제 보육 정보 연계 부재가 지배 쟁점으로 도출. |
| **결론** | 돌봄 안전망의 시간적·공간적 연속성 보장 방향으로 R&R 배분 및 보육 공간 확충이 우선되어야 함. |

> ⚠️ **주의:** 본 논문은 2026년 7월 기준 KCI·RISS·earticle에서 검색되지 않습니다.
> 발표 시 "○○건 분석" 등 구체적 수치를 인용할 경우, 반드시 원문 확인 후 사용하시기 바랍니다.
> 검증 불가 시, 유사 주제의 대체 논문을 권장합니다.

**🔗 대시보드 연계:** 돌봄 공백 해소 중심의 정책 분류 및 부서 라우팅 로직 설계 근거.
