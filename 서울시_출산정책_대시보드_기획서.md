# 서울시 출산·양육 정책 수요 분석 대시보드 기획서

> 4인 팀 프로젝트 병렬 리서치 통합안 (담당: 박채연 — 문제정의·사용자·KPI·통합)

---

## 1. 프로젝트 개요

서울시(또는 자치구) 출산·양육 정책 담당자를 위한, **지역·기간·정책 분야별 시민 수요와 우선순위를 한 화면에서 확인할 수 있는 대시보드**를 기획한다.

---

## 2. 문제정의

```
정책 담당자는 민원(시민제안), 정책별 지원사업 정보, 지역 출산 통계가
서로 다른 채널·부서에 흩어져 있어, 특정 지역에서 어떤 정책 수요가
급증하고 있는지 빠르게 파악하고 우선순위를 정하기 어렵다.

특히 출산·양육 정책은 여성가족실·시민건강국·복지실 등 5개 이상 조직,
10여 개 팀에 대상군(일반/다문화/장애인/위기임산부)별·생애주기별로
쪼개져 있어, 수요를 종합적으로 보는 주체 자체가 없다.
```

## 3. 사용자 페르소나

```
서울시 또는 자치구 여성가족실(저출생담당관 등) 소속
출산·양육 정책 담당자

현재 하는 일: 민원·시민제안, 개별 정책 안내, 자치구 통계를
각각 확인하고 수작업으로 비교·취합

의사결정: 어떤 자치구에 어떤 정책(임신/출산/보육)을
우선 검토·확대할 것인가
```

## 4. AS-IS / TO-BE

```
AS-IS: 담당자가 시민제안·민원, 개별 정책 안내, 자치구 통계를 각각
       확인하고 수작업으로 비교·취합
TO-BE: 지역·기간·정책 분야별 민원(제안) 추이와 수요 우선순위를
       대시보드 한 화면에서 확인
```

## 5. 담당부서 매핑 (문제정의 근거)

서울시 출산·양육 정책 담당업무는 대상군·생애주기별로 아래처럼 파편화되어 있다.

| 유형 | 주요 부서 | 비고 |
|---|---|---|
| 경제적 지원 | 저출생담당관 저출생사업1팀 | 출산급여, 주거비, 산후조리경비 |
| 돌봄 서비스 | 가족담당관 돌봄사업팀 | 아이돌봄, 틈새3종 |
| 임신·건강 관리 | 건강임신지원팀 / 가족건강팀 | 난임치료, 고위험 산모 지원 |
| 위기·취약계층 | 가족지원팀 / 아동보호팀 / 아동복지센터 / 장애인가족지원팀 | 위기임산부, 보호출산, 장애인가정 |
| 다문화 특화 | 다문화지원팀 | 이주여성 건강관리 |

→ 이 파편화가 대시보드 필요성의 핵심 근거. 대상군별 정책 공백(Gap) 발견 기능으로 연결.

## 7. 분석 방법론 (MVP 확정본 — 스코프컷 반영)

당초 임베딩 기반 유사도·클러스터링을 검토했으나, 남은 시간(3~4일) 대비 **키워드 매칭 기반으로 단순화 확정**.

1. **정책분류(category)**: 6개 카테고리(임신/출산/보육/다자녀/위기임산부/다문화)별 키워드 사전 매칭
2. **부서매칭(department)**: 7개 부서별 키워드 사전 매칭, 실패 시 카테고리 기준 대표부서로 자동 보완 (미지정률 78%→5.6%로 개선)
3. **정책 공백 탐지**: VOTE_SCORE(공감수) 상위 + REPLY_YN=N(미답변) 조합으로 "방치된 고수요" 자동 필터링
4. **부정감성 보조신호**: 별도 감성분석 모델 없이 부정어 키워드 매칭으로 경량 처리 (아래 학술 근거 참고)
5. **유사 제안 군집화(클러스터링)**: **보류 상태** — 시간 남으면 TF-IDF+코사인유사도로 추가, 없으면 스킵

### 학술적 근거 (검증 완료)

- **홍향희·이정화(2026)**, "국민신문고 저출산 민원 게시글에 나타난 영유아 정책 내용 분석: CONCOR 분석을 중심으로", 한국생활과학회지 35(1), 1-17. 국민신문고 '저출산' 검색 결과 992건(2006~2022)을 시기별 TF/N-gram/CONCOR로 분석. **이 논문의 실제 빈출 키워드(시험관, 배려석, 워킹맘, 한부모, 미혼부, 양육수당, 우대카드제 등)를 우리 정책분류·부서매칭 키워드 사전에 반영함.**
- **배기련 외(2021)**, "텍스트 마이닝을 활용한 저출산 정책과 대중인식 비교", 디지털융복합연구 19(12), 29-42. 저출산·고령사회 기본계획(정책문서) vs 발표 직후 2주간 네이버 뉴스댓글 25,800건을 비교분석. **CONCOR 분석에서 전 시기 공통으로 검출된 부정 정서 키워드(포기, 노예, 독박, 헬조선)를 우리 부정감성 보조신호 키워드로 채택함.**

**벤치마크 근거(실무 사례)**: 화성시 초거대AI 민원 상담 도입 후 동일 민원 처리시간 42분 단축, 처리건수 10% 향상 사례 존재. K-민원 서비스는 이미 유사 민원 군집화·표준답변 추천 기능을 상용화.

⚠️ **폐기된 참고문헌**: 이전 버전에서 인용했던 "김도우·이경전(2020)", "박서연·최지은(2022)", "송태민 외(2015) — 소셜 빅데이터 기반 저출산 정책 수요 예측"은 검증 결과 **실존하지 않거나(전자 2편) 실제 논문 저자에 가짜 제목을 붙인 것(송태민 2015 — 실제로는 통일인식 관련 논문)**으로 확인되어 전량 폐기함. 방법론 보고서에는 반드시 위 검증된 2편만 인용할 것.

## 10. MVP 화면 구성 (확정)

```
1. 정책 수요 개요       - 총 건수, 미답변율, 급증 키워드, 카테고리 비중
2. 지역별 비교          - 자치구별 제안 건수 + 통계 데이터 (지역미상 다수 주의)
3. 키워드·수요 강도 분석 - 카테고리별 건수 vs 평균 공감도
4. 정책 우선순위 상세   - 공감 높은데 미답변인 것 강조, 담당부서 표시
```

## 11. 팀 통합 시 확정 필요 항목

1. 최종 사용자: 서울시 전체 vs 자치구 담당자
2. 분석 지역: 서울 전체 vs 자치구별
3. 정책 범위: 출산장려 전체 vs 임신·출산·보육 포함
4. 핵심 데이터: 민원(제안) vs 정책목록 vs 지역통계 — 비중 조정
5. 분석 방법: 부서매칭 / 클러스터링 / 수요점수 중 MVP 범위
6. 화면 구성: 위 4개 화면 확정 여부

## 12. 남은 가설 / 검증 필요 사항

- "정책 우선순위를 정할 때 무엇이 어려운지"는 담당자 인터뷰 없이 확인 못한 가설
- 최종 사용자가 서울시 본청인지 자치구인지 자료상 불명확
- 부서 매칭 정확도는 공개 민원사례로 소량 검증 필요 (정답 데이터 부재)
