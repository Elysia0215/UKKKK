## 💡 폴더별 직관적 가이드 (Quick Guide)

| 폴더 명 | 직관적인 역할 설명 | 주요 포함 내용 |
| :--- | :--- | :--- |
| **`docs/`** | 📄 **프로젝트 기획/정의서** | 기획서.md, 분석정의서.docx, 로드맵.txt, 정책자료 |
| **`data/raw/`** | 📊 **수집 원천 데이터** | 상상대로 원문 19,657건, 통계청 출산율/보육시설 현황 |
| **`data/mongttang/`** | 🏛️ **서울시 공식 행정 DB** | 몽땅정보통 323개 사업 DB, 서울시 18개 팀 전화번호 DB |
| **`data/processed/ver1_baseline/`** | 🎯 **1차 수집 기준 (704건)** | 초창기 704건 추출 후보 및 18,953건 제외로그 |
| **`data/processed/ver2_expanded/`** | 🚀 **2차 확장 수집 (824건)** | 생활/돌봄공백 확장 824건 + **팀 검증 전용 137건 파일** |
| **`data/archive/`** | 📦 **과거 압축 백업** | 임시 백업 `.zip` 파일 모음 |
| **`scripts/`** | 🐍 **파이썬 파이프라인** | 19,657건 정제, 분류, 매칭, 웹 크롤링 자동화 코드 |
| **`frontend/`** | 💻 **웹 대시보드** | 서울시 정책 시각화 React/Vite 대시보드 애플리케이션 |
